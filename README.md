# Project Idea Rescue

Project Idea Rescue is a full-stack web application that helps users convert raw ideas into structured, hackathon-ready project plans using a smart backend logic system.

---

## Problem Statement

Students and beginners often struggle to convert vague ideas into complete project structures with features, tech stack, and roadmap.

---

## Solution

This application takes a simple idea and automatically generates:

- Project Title
- Problem Statement
- Key Features
- Suggested APIs
- Tech Stack
- Development Roadmap
- Resume Description

---

## Features

-  AI-like project plan generator (rule-based engine)
-  Instant project title creation
-  Smart feature suggestion based on keywords
-  API recommendations based on idea context
-  Clean and simple UI
-  Full-stack implementation (Frontend + Backend)

---

## How It Works

1. User enters a project idea
2. Backend analyzes keywords
3. System generates structured project plan
4. Frontend displays results in organized sections
